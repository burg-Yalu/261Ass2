部分变量解释：
板子：
boolean batToStopBall：用于游戏开始和粘球技能时，球接触到板子后不反弹，与板子一起移动，直到按空格发射
int batstatus; ：切换板子的变量 //0,1,2,3 means different formats of bats（未实现切换）
球：
double[] distanceX; 用于计算反弹角度，见updateBall()
ballType：切换球的特效（未实现）
ballActive[]:用于判断数组内每个球是否被激活（用于增加球的奖励，待实现）
